# Mind-Seekers Tracker

**Mind-Seekers Tracker** tracks progress toward the Mind-Seeker Feat of Strength — a hidden achievement requiring 32 secret collectibles across mounts, pets, toys, transmog, quests, and achievements.

---

## Features

- **Progress Tracking** — Monitors all 32 secrets: 17 mounts, 9 pets, 2 toys, 1 transmog, 1 quest, 1 achievement
- **Auto-Detection** — Hooks into game events to detect newly obtained secrets automatically
- **Dual-Stage Confirmation** — Items are marked "Detected" (yellow) when obtained, then "Confirmed" (green) when the associated hologram is verified at the Seat of Knowledge
- **Tooltip Scanning** — Automatically detects Record holograms in the Seat of Knowledge by scanning tooltips
- **TomTom Integration** — One-click waypoints for secret locations
- **Collapsible UI** — Compact tracker window that can collapse to just a title bar
- **Status Banner** — Alerts when 17+ secrets are collected and you're ready to visit the Pearl of the Abyss

---

## Usage

Toggle the tracker:

```
/mst
/mindseeker
```

Additional commands:

- `/mst reset` — Reset tracker position to default
- `/mst debug` — Print API results for all secrets to chat

---

## Installation

Download and extract into:

```
World of Warcraft/_retail_/Interface/AddOns/MindSeekersTracker
```

Restart WoW or `/reload`.
